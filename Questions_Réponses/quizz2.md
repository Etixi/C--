**Question 10 :**
Qu'affichera le code suivant ? 

```
int three = 3;
cout << “There are “ << “three” << “ stooges” << endl;
cout << “Curly is my favorite of the ” << three << endl;
```

```

Curly is my favorite of the 3
```
